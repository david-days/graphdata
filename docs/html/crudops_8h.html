This header is for basic graph structure operations (Create,Read,Update,Delete and similar).

The functions specified here are for general graph structures and data, for use in various other implementations. These are the most general operations, dealing with basic graph structures (nodes, edges, dimensional data, etc.). For implementation-specific operations, use the XXXgraph and XXXops headers under the impl folders.
