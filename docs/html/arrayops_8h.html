int arrayResetGraph ( struct graph_t *  g,
void *  args,
void(*)(void)  callback 
)

Implementation to "reset" the graph according to the given argument pointer.

For some implementations, it is more efficient to reuse the existing graph structure and perform a "zero-out" of the data, rather than rebuilding from scratch. This function pointer provides that option.

Parameters
gGraph structure to be zeroed or modified according to reset logic
argsArguments to be used in the reset process, if necessary
callbackCallback to be executed when graph has been reset.
Returns
0 if there was an error during the reset; 1 if the reset completed;
