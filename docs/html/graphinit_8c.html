Create a dimension structure containing the given values in order (x, y, z, etc)

This is used to initialize (and contain) the ordered size of the spatial dimensions given. Consumers are responsible for calling free() on the structure, or passing it to a standard clearing function

Parameters
dimszNumber of dimensions
...Additional dimension values passed
Returns
Pointer to a dimensions_t structure that holds the count and values of the structure, or a null pointer if there is a problem allocating the memory.
