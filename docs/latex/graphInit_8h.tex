Initialize a graph according to the flags set in the GRAPHDOMAIN value. 

Using the flags set in the typeflags parameter, create a graph structure capable of supporting the requirements in an efficient (memory or Big-\/O) manner.

The parameter gtype holds the flags for what graph is desired. For example, For a directed graph, used to represent a set of cartesian coordinates and no extra label nodes, pass in the value created by DIRECTED \texorpdfstring{$\vert$}{|} LINKED \texorpdfstring{$\vert$}{|} SPATIAL \texorpdfstring{$\vert$}{|} UNLABELED.

If an empty set of flags are passed to the typeflags parameter, the returned graph will be UNDIRECTED \texorpdfstring{$\vert$}{|} LINKED \texorpdfstring{$\vert$}{|} GENERIC \texorpdfstring{$\vert$}{|} UNLABELED.


\begin{DoxyParams}{Parameters}
{\em gtype} & Type of graph implementation to be created. Flag values set underlying structures and metadata. \\
\hline
{\em lblcount} & Number of label nodes to be used within the graph. Required for LABELED flag; ignored for all others. \\
\hline
{\em dims} & Dimensional parameters structure. Required for ARRAY graphs; otherwise, may be NULL. The returned graph will hold the reference to the structure that was passed. \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
If successful and valied, initialized graph structure, according to the flags. Otherwise, a NULL pointer.
